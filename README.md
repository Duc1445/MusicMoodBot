# Music Mood Bot

Một ứng dụng giúp gợi ý nhạc phù hợp với cảm xúc của người dùng. Ứng dụng sử dụng Python cùng Flet ở phía giao diện, còn phía backend xử lý các thuật toán học máy để phân loại tâm trạng và đưa ra gợi ý.

## Công nghệ sử dụng

- Python phiên bản 3.10.11
- Flet để tạo giao diện
- Virtual Environment để quản lý thư viện

## Cách tổ chức thư mục

- frontend: Chứa mã giao diện người dùng
- backend: Chứa logic xử lý, phân tích ngôn ngữ, và gợi ý nhạc
- docs: Tài liệu dự án CDIO
